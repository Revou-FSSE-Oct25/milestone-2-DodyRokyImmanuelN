Tentang RevoFun

RevoFun adalah studio game yang berfokus pada mini game berbasis web yang ringan, cepat dimainkan, dan cocok untuk mengisi waktu luang.

Website ini dibangun sebagai proyek pembelajaran JavaScript, menampilkan beberapa game sederhana dengan fokus pada interaktivitas dan pengalaman pengguna.

Misi

Membuat game kecil yang seru sekaligus menjadi sarana belajar logika dan pemrograman.

Teknologi

Dibangun dengan HTML, Tailwind CSS, dan JavaScript/TypeScript untuk mengelola interaksi dan game logic.

Fokus Belajar

Melatih DOM manipulation, event handling, dan problem solving lewat implementasi beberapa mini game.
